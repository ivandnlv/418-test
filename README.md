# UI компонент Box

В данном небольшом задании используются следующие технологии:
- React;
- Typescript;
- React Context;
- Scss modules;
- Vite;

## Запуск проекта:

Если нет yarn'а:
```
npm install -g yarn
```
Далее:
```
yarn
```
```
yarn dev
```
Пропсы компонента Box:

| name       | type               | required |
| ------------- |:------------------:| -----:|
| children     | ReactNode[] or ReactNode    | true |
| headerAddon     | ReactNode[] or ReactNode |   false |
| footerAddon  | ReactNode[] or ReactNode         |    false |
